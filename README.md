# qtc
Open-Source Quantum Topological Computer you can build at home.

To start, a single pair of entagled electrons, with 48 modifiable properties each ( -4 to +4 OAM and 0 to 5 concentric rings, radial degrees of freedom ) , we can enconde the entire state matrix as a manifold directly in the relationship of the entagled photons.

Much of this works stems directly from [Revealing the topological nature of entangled orbital angular momentum states of light](https://doi.org/10.1038/s41467-025-66066-3)
